# Currency Converter

Автор: *нафикова*

## Описание
Простое GUI-приложение для конвертации валют с сохранением истории.

## Требования
- Python 3.x
- Библиотеки: `tkinter`, `requests`

## Установка и запуск
1. Установите зависимости:
   ```
   pip install requests
   ```
2. Получите API-ключ на [exchangerate-api.com](https://www.exchangerate-api.com/).
3. Вставьте ключ в переменную `API_KEY` в коде.
4. Запустите:
   ```
   python main.py
   ```

## Примеры использования
- Введите сумму, выберите валюты и нажмите «Конвертировать».
- История сохраняется в `history.json`.
