# Currency Converter 💱

**Автор:** *Нафикова*
**Дата:** 01.05.2026

## Описание проекта

Простое кроссплатформенное приложение для конвертации валют с графическим интерфейсом (Tkinter), использованием внешнего API для актуальных курсов и сохранением истории операций в файл.

## Требования

-   **Python 3.8+**
-   Библиотеки: `tkinter` (входит в стандартную библиотеку), `requests`

## Установка и запуск

1.  **Установите Python** (если не установлен).
2.  **Установите зависимости:**
    ```bash
    pip install requests
    ```
3.  **Получите API-ключ** на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
4.  **Вставьте ключ** в файл `main.py` в строку `API_KEY`.
5.  **Запустите приложение:**
    ```bash
    python main.py
    ```
6.  **История** сохраняется в файл `history.json`.

## Использование

1.  Выберите валюту «Из» и «В».
2.  Введите сумму (только положительное число).
3.  Нажмите «Конвертировать».
4.  Результат появится под кнопкой.
5.  Все операции сохраняются в таблицу истории.
